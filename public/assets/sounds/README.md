# Audio Assets

이 폴더에 오디오 파일을 추가하세요.

## 필요한 파일

### BGM (배경음악)
- `bgm-act1.mp3` - Act 1 (Day 1-5): 따뜻하고 설레는 분위기
- `bgm-act2.mp3` - Act 2 (Day 6-20): 긴장감이 쌓이는 분위기
- `bgm-act3.mp3` - Act 3 (Day 21-30): 미니멀하고 압박감 있는 분위기

### 효과음
- `click.mp3` - 버튼 클릭 소리
- `complete.mp3` - 작업 완료 소리
- `bug.mp3` - 버그 발견 소리
- `warning.mp3` - 경고 소리

## 현재 상태

오디오 시스템은 설정되어 있지만, 실제 파일은 없습니다.
위 파일명으로 오디오를 추가하면 자동으로 작동합니다.

파일이 없어도 게임은 정상적으로 플레이 가능합니다.
